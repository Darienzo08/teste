1. crie um repositorio no github
2. no prompt de comando direcione para a pasta que voce deseja
	(cd[change directory]) "pasta desejada" ou "..(para voltar)".
3. coloque o comando git clone "link".git(para clonar)
4. salve todos os arquivos dentro da pasta clonada
5. no prompt de comando, coloque o comando git add
6. coloque o comando git commit -m "nome do commit"
7. agora, para mandar o conteudo da pasta para o github, de o comando git push
	agora o conteúdo da pasta está no github
